English & Korean language based UI Kit for Sketch. And it’s FREE!

A Google Material Design based UI Kit, with “icons”, “buttons” and “toast”. So much “toast” you could feed a country! Symbols within symbols within symbols for easy-to-use overrides.

If you want to know more about this design, check this Medium post and any feedback would be much appreciated!
